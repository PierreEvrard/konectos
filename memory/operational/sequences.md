# Séquences multi-touch (cross-platform)

Documente ici les séquences actives (LI → WhatsApp → Instagram, etc.). Chaque ligne ou bloc = une séquence nommée.

## Format recommandé

### Nom de la séquence

- **Déclencheur** : [À CONFIGURER]
- **Étapes** :
  1. [Plateforme] — Action — Délai
  2. …
- **Statut** : brouillon | active | en pause
- **Airtable** : pas de table Séquences — si tu dois lier un contact à une étape, utilise le champ **Notes** ou **Statut** sur **Contacts**, ou un tag texte libre.

---

## Exemple (template)

### Nouveau lead LinkedIn → WhatsApp

1. LinkedIn — invitation ou message (`to`) — J0
2. LinkedIn — relance — J+3
3. WhatsApp — premier message (numéro obtenu) — J+7
