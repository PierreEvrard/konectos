# Weekly — Revue hebdo

Combine **bilan**, **mémoire** et **planning** pour la semaine suivante.

## Quand activer

- « weekly », « revue hebdo », « rituel du lundi »

## Workflow

### 1. Bilan chiffré court

Utiliser les appels de `/report` (usage + queue failed + aperçu CRM) sur **7 jours** ou **30 jours** selon demande.

### 2. Qualitatif

Questions rapides à l’utilisateur (optionnel) :

- Meilleure conversation de la semaine ?
- Message type qui a le mieux marché ?

### 3. Mémoire

Ajouter une entrée datée dans `memory/synthesis/weekly-synthesis.md` :

- Ce qui a marché / pas marché
- Décisions
- Focus semaine prochaine

### 4. Priorités S+1

Lister **3 tâches** max (ex. « finaliser séquence WA », « 10 icebreakers IG », « ajuster fenêtre d’envoi »).

### 5. Suite

Proposer `/memory-save` pour lessons détaillées si besoin.
