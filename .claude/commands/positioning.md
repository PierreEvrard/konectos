# Positionnement & messaging

Clarifie la **proposition de valeur** et le **ton par canal** pour alimenter `brand.md`, `offer.md` et les prompts dans `agent-prompts.md`.

## Quand activer

- « positionnement », « messaging », « comment parler de mon offre »

## Prérequis

Lire `memory/identity/brand.md`, `memory/identity/offer.md`, `memory/operational/agent-prompts.md`.

## Workflow

1. Synthèse en 1 phrase : pour qui / quel problème / quelle promesse (réaliste).
2. Mettre à jour `memory/identity/brand.md` (ton, formulations à privilégier / éviter).
3. Vérifier cohérence `offer.md` (CTA, règles prix).
4. Ajuster les blocs **Identity / Context** dans `agent-prompts.md` pour LI / WA / IG si besoin.
5. Proposer `/icebreaker` pour tester le messaging.
