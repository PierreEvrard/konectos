# ICP — Profil client idéal

Met à jour **`memory/identity/persona.md`** (profil + ICP unifiés).

## Quand activer

- « ICP », « cible », « persona », « qui contacter »

## Prérequis

Lire `memory/identity/persona.md` et `memory/identity/offer.md`.

## Workflow

1. Poser 3–5 questions courtes (rôle cible, secteur, taille, signaux d’intérêt, anti-personas) si infos manquantes.
2. Rédiger les sections **Profil utilisateur** et **ICP** + **Critères de scoring** (Tier 1/2/3).
3. Écrire le fichier (remplacer `[À CONFIGURER]` par du concret).
4. Suggérer `/prospect` ou `/score` pour appliquer le nouvel ICP.
